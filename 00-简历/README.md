# 1. 简历上要有的内容
## 1.1. 个人信息
  - 姓名、性别、出生日期、联系电话、邮箱、地址
## 1.2. 教育信息
  - 毕业院校、起止年份、专业、学业成绩排名、主修课程、排名
    + 吉首大学、2018~至今、本科、软件工程专业、
## 1.3. 实习信息
  - 何时、何地、什么职位、做什么事、做得如何
## 1.4. 活动经历（校园经历一般不建议写）
  - 何时、何地、什么职位、做什么事、做得如何
## 1.5. 技能与评价
  - 相关技能（有关、有效、有证明），自我评价用数据
  - 技能+兴趣

# 2. 格式
  - 一张A4内
  - PDF格式
  - 准备多份简历，应对不同公司
  - 文件命名：名字-应聘职位-联系方式
